🛒 NexaCart - Full-Stack E-Commerce Platform
A modern, responsive e-commerce web application featuring a sleek UI, real-time database integration, and secure payment processing.

🚀 Overview
NexaCart is a functional e-commerce storefront built to demonstrate a seamless integration between a vanilla frontend and a "Backend-as-a-Service" (BaaS) architecture. It handles everything from user authentication to real-time product fetching and secure checkouts.

✨ Key Features
User Authentication: Secure Sign-up and Login system powered by Supabase Auth.

Dynamic Storefront: Products are fetched dynamically from a PostgreSQL database.

Persistent Cart: A fully functional shopping cart that retains items using LocalStorage.

Payment Gateway: Integrated Razorpay API for processing secure payments (Test Mode).

Responsive Design: Optimized for Desktop, Tablet, and Mobile views.

🛠️ Tech Stack
Frontend: HTML5, CSS3, JavaScript (ES6+)

Backend/Database: Supabase (PostgreSQL)

Payments: Razorpay SDK

Styling: Custom CSS (Flexbox & Grid)# Full-Stack-E-Commerce-Platform
